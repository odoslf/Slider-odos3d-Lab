# Soporte / Atención

Para ayudarte rápido, incluye SIEMPRE:

1) Versión (v1.0 / v1.1 / etc.)
2) Pieza exacta (nombre del STL/STEP)
3) Material y parámetros de impresión (altura capa, perímetros, relleno, soportes)
4) Fotos: súbelas primero a `images/inbox/` (cumpliendo `PRIVACY.md`)
5) Si es GRBL: pega tu configuración `$$` y explica el fallo

## Dónde preguntar
- Issues: bugs, roturas, piezas defectuosas, docs incorrectas
- Discussions: dudas, montaje, recomendaciones y variantes
