# Compatibilidad

## Objetivo
Dejar claro qué piezas y ajustes van con qué configuración.

## Tabla (rellenar)
- Hardware: v1.0
  - Piezas: `prints/STL/v1/...`
  - GRBL: `grbl/recommended_settings.md`
  - Límites: `docs/limits.md`

## Notas
- Si una pieza cambia dimensiones críticas → subir versión (v1.1 / v2.0).
- Si cambia recorrido real → actualizar límites y GRBL ($130).
