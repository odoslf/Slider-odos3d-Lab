# Documentación interna del sitio público

Esta carpeta conserva notas de mantenimiento, checklists y mapas de subida que no forman parte de la web pública principal.

Reglas:
- No usar `permalink`.

- Las rutas públicas antiguas deben redirigir a páginas útiles del sitio.
