# Checklist de publicación

## Checklist — Activar GitHub Pages
- [ ] Repo público o plan que permita Pages en privado
- [ ] Settings → Pages configurado (main /docs)
- [ ] Verificar que la URL funciona
- [ ] Abrir `privacy-policy` en navegador y revisar el contenido

## Checklist — Google Play (cuando toque)
- [ ] Pegar URL de `privacy-policy` en Play Console
- [ ] Confirmar modelo: compra única, sin anuncios
- [ ] Revisar texto de privacidad dentro de la app (About) y enlazar a la URL
