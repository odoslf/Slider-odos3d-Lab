FAQ

Preguntas frecuentes

¿Necesito un slider para usar Smart Timelapse AI?

No. Puedes empezar solo con el móvil y añadir movimiento más adelante cuando lo necesites.

¿La app funciona con hardware GRBL?

Sí. Está pensada para trabajar con sliders GRBL compatibles dentro del mismo flujo de captura y control.

¿El control remoto funciona por Internet?

No. Funciona en red local y se protege con PIN para uso dentro de la misma conexión.

¿En modo DSLR el móvil hace las fotos?

No. La cámara hace las fotos y la app coordina el flujo para mantener la secuencia bajo control.
