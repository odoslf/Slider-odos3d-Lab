# Ajustes recomendados de impresión (v1)

## Material
- PLA: prototipo / interior
- PETG: recomendado para uso real (mejor resistencia/temperatura)

## Parámetros base
- Boquilla: 0.4 mm
- Altura de capa: 0.2 mm (0.16 si quieres más calidad)
- Perímetros: 3–4
- Top/Bottom: 4–6
- Relleno: 20–35% (según pieza)
- Soportes: solo donde se indique
