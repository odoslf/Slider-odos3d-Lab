## Mechanical checklist

- Belt tensioned without excess.
- Pulleys aligned.
- Carriage moves without rubbing.
- Screws checked.
- Full travel is clear.

## If you use GRBL

- Save `$$` before changing parameters.
- Set `$130` to the real travel length.
- Test short movements first.
- Reduce speed or acceleration if steps are lost.
