## Preparation

Review the BOM, print the required parts, and prepare basic tools: Allen keys, screwdriver, caliper, and cleanup tools.

## Recommended order

1. Assemble side supports and main structure.
2. Install rods or rails and check alignment.
3. Assemble the carriage.
4. Install pulleys, belt, and tensioner.
5. Install the camera or phone mount.
6. Mount motor and electronics.
7. Check that motion is smooth across the full travel.

## Final verification

Before using the slider in a real session, check screws, belt tension, play, free travel, and mounted equipment safety.
