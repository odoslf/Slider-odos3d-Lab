1. Print the v1 parts from `prints/STL/v1/`.
2. Prepare the rods or rails and remove burrs.
3. Assemble the carriage and check for smooth travel.
4. Install GT2 pulleys and belt with moderate tension.
5. Mount the motor and verify alignment.
6. Install the recommended electronics.
7. Set driver current before long movements.
8. Connect the endstop if your build uses one.
9. Flash GRBL 1.1 and apply settings for your travel length.
10. Run a short movement test before using the full travel.
