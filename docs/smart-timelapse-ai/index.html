Slider por Bluetooth

Movimiento con precisión sobre GRBL

Conecta sliders compatibles y trabaja con escenas, keyframes y planificación de movimiento dentro de un flujo más avanzado.

  • GRBL 1.1 recomendado.
  • 11 escenas, incluida Producto 360 calibrable.

Flujo DSLR

La cámara captura. La app coordina.

El móvil gestiona el flujo y acompaña el proyecto. La cámara guarda las fotos en su tarjeta y la app continúa el proceso con material compatible.

  • Disparo automático mediante relé M3/M5.
  • Perfecto para intervalos largos y secuencias delicadas.
  • RAW no se procesa en la app.

Control remoto local

Un panel web real para controlar el slider desde otro dispositivo.

La app puede levantar un panel remoto en red local para abrirlo desde otro móvil, tablet o PC en la misma Wi‑Fi o hotspot. No es una promesa genérica de remoto: es una interfaz web pensada para trabajar el set sin tocar el teléfono principal.

Cómo funciona

Panel web en LAN con PIN

La app genera una URL del panel remoto y la protege con PIN. El acceso está pensado para red local y para sesiones de trabajo reales.

  • Móvil y cliente remoto en la misma Wi‑Fi o hotspot.
  • PIN obligatorio.
  • URL para copiar y abrir desde otro dispositivo.

Exportación y edición

Todo lo que puedes hacer al cerrar el proyecto.

La exportación es una de las partes fuertes del producto. La app permite transformar la secuencia en una salida final utilizable sin salir del mismo ecosistema.

Resolución y formato

MP4 o GIF, desde 720p hasta 4K

  • Exportación en 720p, 1080p o 4K UHD según dispositivo.
  • Si 4K no está disponible o el equipo no llega, la app avisa y usa fallback a 1080p.
  • GIF para secuencias cortas y salida rápida.
  • MP4 para piezas largas o publicación final.

Perfiles y composición

Original, 9:16 y 1:1

  • Perfil Original para mantener el encuadre base.
  • 9:16 para Reels, TikTok y Shorts.
  • 1:1 para cuadrado e Instagram Feed.
  • FPS configurable para decidir la velocidad final.

Edición integrada

Ajustes útiles de verdad

  • Texto y timestamp.
  • Música o audio con punto de inicio.
  • Trim y control de velocidad.
  • Reversa, rotar y voltear.
  • Loop y fade in/out.

Compatibilidad del flujo

Del material al vídeo final

En modo DSLR, la cámara guarda las fotos en su tarjeta. Si luego importas material compatible al móvil, la app puede continuar el flujo de edición y exportación aquí.

Preguntas frecuentes

Respuestas claras para empezar sin dudas.

Cuando el producto se entiende rápido, la decisión también llega antes.

¿Necesito un slider para usar Smart Timelapse AI?

No. Puedes empezar solo con el móvil y usar SmartCapture AI, el editor integrado y la exportación sin hardware externo.

¿La app funciona con hardware GRBL?

Sí. Está pensada para sliders GRBL compatibles, con GRBL 1.1 como referencia recomendada para una experiencia más sólida.

¿El control remoto funciona por Internet?

No. El panel remoto funciona en red local, dentro de la misma Wi‑Fi o hotspot, con protección por PIN.

¿Qué puedo hacer exactamente al exportar?

Puedes generar MP4 o GIF, usar perfiles Original / 9:16 / 1:1, añadir audio, texto o timestamp, ajustar velocidad, hacer trim, usar reversa, rotar, voltear y exportar hasta 4K según dispositivo.
