# Seguridad

Este proyecto implica hardware y cargas (cámara/móvil). Si detectas:
- roturas con riesgo,
- sujeciones inseguras,
- piezas que puedan provocar caída de equipo,
- recomendaciones de montaje peligrosas,

abre un Issue con:
- versión (vX),
- pieza exacta,
- escenario,
- fotos técnicas (según `PRIVACY.md`).
